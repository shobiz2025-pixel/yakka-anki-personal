# 薬価算定の基準 暗記ツール

令和6年度・令和8年度の「薬価算定の基準」をもとにした暗記・学習ツールです。

🔗 **https://shobiz2025-pixel.github.io/yakka-anki/**

## 機能

- 令和6年度版 / 令和8年度版の切り替え
- 解説・Q&A・関連スライドの閲覧
- ○△× による習熟度チェック
- 苦手リスト・閲覧ランキング・学習カレンダー（ダッシュボード）
- キーワード検索・章フィルター
- PWA対応（ホーム画面追加可能）

## 使い方

上記URLをブラウザで開くだけで動作します。インストール不要。

## データ

- 令和6年度版：`data.js` / `js/qa_text_01〜07.js`
- 令和8年度版：`data/r08/data.js` / `js/qa_text_r8_01〜08.js`
